pudb

  • view local variables
  • create watches
  • jump into ipython shell
  • enter exceptions!
  • jump through stack

Get started in two steps:

  • pip install pudb
  • python -m pudb.run script.py

    • or

  • from pudb import set_trace; set_trace()

cheat sheet

  • b - breakpoint
  • s - step into
  • n - next statement
  • c - continue
  • t - teleport (jump to)
  • q - quit
